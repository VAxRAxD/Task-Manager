# Task-Manager
A Command Line Interface application build using Python3 and its library Typer
- ./app.py add PRIORITY "TASK"
- ./app.py ls 
- ./app.py del INDEX 
- ./app.py done INDEX 
- ./app.py help 
- ./app.py report
